# proyecto_godot_2526
Repositorio para el proyecto de Godot del curso 25-26 Con Lóxica.

# Convenciones
Los nombres de carpetas y archivos, deben utilizar snake_case.
Ejemplo:
	carpeta1
	player.tscn
	player.gd

Los nombres de los nodos raíz de las escenas, deben ser iguales
a los del archivo, pero en PascalCase.
Ejemplo:
	player.tscn
	Player (nodo raíz)
